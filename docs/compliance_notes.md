# Compliance Notes

- Spotify and YouTube are metadata-only in this project. Do not download, cache, or extract commercial audio from them.
- Jamendo and Freesound results may be imported only when the selected track/sound license and provider terms allow it.
- User recordings are private task inputs by default and must not be added to training data without explicit opt-in consent.
- Rubber Band, Demucs, torchcrepe/RMVPE, model weights, and binary packaging require separate license review before commercial distribution.
