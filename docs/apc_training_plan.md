# Long-Term APC Training Plan

Self-trained APC is intentionally not implemented in the first upgrade phase.

Future work:
1. Define opt-in data consent and deletion policies.
2. Curate legally usable training data; do not use third-party platform audio without explicit rights.
3. Prototype offline research code under `research/apc_prototype/`.
4. Compare APC output against the DSP renderer with objective pitch/timing metrics and listening tests.
5. Add model cards, licensing manifests, and resource requirements before production use.
